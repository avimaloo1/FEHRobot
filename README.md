# FEHRobot
Designed, built, and developed a robot that followed the Brutus Garden course, which involved picking up and placing an apple bucket on a table, rotating a compost bin, clicking the humidifier button, opening and closing a window, and flipping a lever all within 2 minutes. 

Completed in 1 minute and 56 seconds, and received all primary and secondary points. Placed Top 16 in the Ohio State Engineering Showcase
